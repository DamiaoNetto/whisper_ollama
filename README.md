Projeto: Transcrição e Sumarização de Áudio com Whisper e Ollama

Este projeto utiliza o modelo Whisper para transcrição de áudio e o modelo Ollama para gerar um resumo baseado no texto transcrito. Ele é ideal para processar arquivos de áudio e obter uma versão resumida do conteúdo.

Funcionalidades

Transcrição de áudio: Utiliza o modelo Whisper para converter áudio em texto.

Geração de resumo: Usa o modelo Ollama para resumir o texto transcrito.

Template personalizável: Permite definir o formato do prompt enviado ao modelo de linguagem.

Requisitos

Python 3.8 ou superior.

Dependências listadas no arquivo requirements.txt.

Ferramenta externa FFmpeg para processar áudio.
